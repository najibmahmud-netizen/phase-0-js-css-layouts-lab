### Cat Breeds Layout Project

A simple responsive web page that displays different cat breeds using Flexbox and CSS media queries. The project demonstrates how to style cards, create responsive layouts, and apply hover effects.
**Author:** Najib Mahmud

## Features
Responsive layout using Flexbox
Media queries for mobile-friendly design
Styled cat breed cards with:
Images
Headings
Descriptions
Hover effect with smooth scaling (transform: scale(1.05))
Clean and simple UI design

## Pages Overview



This project consists of a single webpage:

- **Home Page (index.html)** - Displays a collection of cat breeds using a card layout. Each card includes an image, a title, a short description, and a "Learn More" link. The layout is built using Flexbox and adapts to smaller screens using media queries for a responsive design.

## Technologies Used

This project was built using:

- **HTML5** - Structure of the webpage 
- **CSS3** - Styling and layout 
- **Flexbox** - Responsive layout design 
- **Media Queries** - Mobile responsiveness 
- **Jest** - Testing framework for CSS validation
